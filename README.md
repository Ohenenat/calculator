# Advanced Calculator

A feature-rich calculator application built with HTML, CSS, and JavaScript.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division, and modulo
- **Decimal Support**: Perform calculations with decimal numbers
- **History Tracking**: View and click on previous calculations to reuse results
- **Multiple Themes**: Switch between Light, Dark, and Blue themes
- **Keyboard Support**: Use your keyboard for faster calculations
- **Error Handling**: Division by zero detection and error messages
- **Responsive Design**: Works on different screen sizes

## How to Use

1. Open `index.html` in your web browser
2. Click number buttons or use your keyboard to enter numbers
3. Click an operation button to select your operation (+, −, ×, ÷, %)
4. Enter the second number
5. Click equals (=) or press Enter to calculate
6. View your calculation history on the right side

## Keyboard Shortcuts

- **Number Keys**: 0-9 to enter numbers
- **+, -, *, /**: Perform operations
- **%**: Modulo operation
- **.**: Add decimal point
- **Enter** or **=**: Calculate result
- **Backspace**: Delete last digit
- **Escape**: Clear all (AC)

## Theme Selection

Switch themes using the dropdown in the header:
- **Light**: Default light theme
- **Dark**: Dark mode for comfortable viewing
- **Blue**: Blue-themed interface

Your theme preference is saved in your browser.

## File Structure

- `index.html` - Main HTML structure
- `script.js` - Calculator logic and event handling
- `styles.css` - Styling and theme definitions
- `README.md` - Documentation
